--Nome do software: IRPFparaExcel

 * Descrição: Este programa obtém os dados de clientes cadastrados no programa IRPF da Receita Federal do Brasil e gera um arquivo do excel (.xls) com os mesmos.

 * Description: This program obtains the data of clients registered in the Brazil Federal Revenue Service's software and generates an excel file (.xls) with them.
		
 

--Pré-requisitos

 * Python 3.6
 * Módulo do Python openpyxl  

--Módulos não nativos do Python utilizados:

	*openpyxl:
		 Autores: Eric Gazoni, Charlie Clark
	         Site do projeto: https://openpyxl.readthedocs.io/en/stable/
